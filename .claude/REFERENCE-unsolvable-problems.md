# Unsolvable Problems

**Definition:** simulation games are problem-solving games; a genuinely interesting problem is fundamentally unsolvable. The depth of a simulation scales roughly with how many sources of unsolvability it activates simultaneously.

**Partitioning axis: the *source* of unsolvability**, not the problem itself. Any given problem typically stacks multiple sources.

---

Seven sources of unsolvability. A genuinely interesting problem usually draws on several at once.

1. Logical Impossibility — No solution exists in principle; the problem violates formal structure. → Halting problem, Gödel incompleteness, CH in ZFC, Rice's theorem, Arrow's impossibility, liar paradox.
2. Epistemic Opacity — A solution exists but the information required is unavailable or unknowable (in the sharpest cases — quantum indeterminacy — the fact doesn't exist until extracted, not merely hidden). → Poker (hidden cards), fog of war, quantum indeterminacy, chaotic systems, other minds.
3. Computational Intractability — Information is fully present but the search space is too large to traverse. → Chess, Go, NP-hard optimization, protein folding, curse of dimensionality.
4. Dynamic Instability — The problem mutates faster than you can solve it (includes adversarial agents as the sharpest case). → StarCraft, markets, Red Queen races, evolving ecosystems, arms races.
5. Value Pluralism — No single objective exists; the dimensions are incommensurable. → Civilization (score vs. culture vs. military), trilemmas, moral dilemmas, aesthetic judgment.
6. Reflexive Closure — The act of attempting a solution modifies the problem. → Goodhart's law, observer effects, metagames, self-fulfilling/defeating prophecies, Lucas critique.
7. Semantic Underdetermination — The question itself lacks crisp meaning; the answer space has no well-defined boundary. → Sorites ("when is a heap a heap?"), "is this art?", category errors, "what counts as winning?"

MECE cut is by the **source** of unsolvability, not the problem itself — any given game typically stacks multiple. Chess = (3)+(4). Poker = (2)+(4). Civ = (3)+(4)+(5). Dwarf Fortress = (3)+(4)+(5)+(6). The depth of a simulation scales roughly with how many of these axes it activates simultaneously.
